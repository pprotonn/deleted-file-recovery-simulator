# Deleted File Recovery Simulator

## 📌 Project Overview
This project simulates how deleted files can be recovered in a computer system.

## 🧠 Concept
- Files are stored in disk blocks
- Deleting a file marks blocks as FREE
- Recovery is possible if data is not overwritten

## ⚙️ Features
- Create file
- Delete file
- Recover file
- Disk visualization
- GUI using Tkinter

## 🛠️ Technologies Used
- Python
- Tkinter

## 🚀 How to Run
```bash
python gui.py
